ChinaTradeResolve AI Assistant v3.1

Staging package with six interface languages, a public multilingual AI information assistant, application triage, case status pages and voluntary crypto support.

The AI widget is server-controlled and remains hidden until ENABLE_AI_ASSISTANT, OPENAI_API_KEY and a model are configured. The API key is never exposed to browser code.

See README.md, README_RU.txt and AI_ASSISTANT_SETUP_RU.md.
Never upload .env, API keys, passwords, seed phrases or private keys to GitHub.
