ChinaTradeResolve v3.7.2

- `/ready` теперь проверяет реальное подключение к базе данных и возвращает HTTP 503 при её недоступности.
- `/health` остаётся liveness-проверкой и не падает при временном отказе базы; зависимые показатели возвращаются как `null`.
- Добавлен GitHub Actions CI: проверка зависимостей, компиляция, полный набор тестов и аудит runtime-зависимостей.
- Добавлены регрессионные тесты для отказа базы данных.
