ChinaTradeResolve Multilingual v2.0

Готовая многоязычная версия сайта для тестового развёртывания.

Поддерживаемые языки:
- английский;
- французский;
- немецкий;
- испанский;
- русский;
- сербский.

Что реализовано:
1. Автоматическое определение языка браузера при первом посещении.
2. Ручной переключатель языка на компьютере и телефоне.
3. Запоминание выбранного языка в браузере.
4. Перевод главной страницы, формы, страницы статуса, страницы поддержки,
   юридических страниц и примера оценки дела.
5. Синхронизация языка интерфейса с полем «Предпочитаемый язык» в заявке.
6. Локализованные сообщения проверки формы и отправки заявки.
7. Локализованные автоматическая оценка, статус дела и письма клиенту.
8. Сохранено исправление формы v1.8: ошибка [object Object] пользователю не показывается.
9. Telegram и WhatsApp не добавлялись — это оставлено на следующий этап.

Проверка:
- 14 автоматических тестов из 14 пройдены;
- проверены английская, французская, немецкая и испанская локали браузера;
- отдельно проверены мобильный переключатель, французская форма,
  страница поддержки и политика конфиденциальности.

Запуск локально:
1. Скопируйте .env.example в .env.
2. Задайте отдельные надёжные значения APP_SECRET и ADMIN_TOKEN.
3. Установите зависимости: python -m pip install -r requirements.txt
4. Запустите: uvicorn app.main:app --reload
5. Откройте: http://127.0.0.1:8000

Для Render используйте Docker-развёртывание и постоянную PostgreSQL-базу через DATABASE_URL.
Не загружайте .env, пароли или другие секреты в GitHub.

Важно перед публичным запуском:
- заменить временные сведения об операторе;
- провести юридическую проверку Privacy, Terms, AI Notice и Disclaimer;
- подключить рабочую отправку электронной почты;
- проверить резервное копирование и мониторинг;
- снять noindex только после готовности публично индексировать сайт.
